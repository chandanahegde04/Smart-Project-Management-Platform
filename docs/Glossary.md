# Glossary

## Frontend

The part of the application users interact with.

Example:
React Application

---

## Backend

The server-side application that contains business logic.

Example:
ASP.NET Core API

---

## API

A mechanism that allows two applications to communicate.

Example:
React calling ASP.NET Core.

---

## Database

Permanent storage for application data.

Example:
PostgreSQL

## Browser

Software used to access websites.

Examples:
Chrome, Safari, Firefox.

---

## Server

A computer that provides data to clients.

---

## HTTP

Protocol used for communication between browser and server.

---

## Request

Message sent from client to server.

---

## Response

Message sent from server to client.
