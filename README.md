# Smart Project Management Platform

A modern enterprise-grade project management platform built using React, ASP.NET Core, PostgreSQL, Docker, and Azure.

## Overview

This application enables organizations to manage projects, tasks, team members, and project progress through a centralized platform.

The project is being developed as part of a structured transition from desktop application development (WPF/.NET) to modern Full Stack Engineering.

## Key Features

- User Authentication & Authorization
- Role-Based Access Control
- Project Management
- Task Management
- Team Collaboration
- Activity Tracking
- Dashboard & Analytics
- AI-Powered Sprint Summaries
